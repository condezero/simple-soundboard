# Simple Soundboard

An interactive soundboard module for Foundry VTT with configurable buttons to play sound effects and music during your sessions.

## Features

- Configurable sound buttons
- Easy-to-use interface
- Multi-language support (English, Spanish)

## Installation

1. In Foundry VTT, go to **Add-on Modules**
2. Click **Install Module**
3. Paste the manifest URL or search for "Simple Soundboard"
4. Click **Install**

## Compatibility

- **Minimum**: Foundry VTT v11
- **Verified**: Foundry VTT v12
